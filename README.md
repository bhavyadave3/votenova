# VoteNova

A modern, interactive voting web app built using Flask + JavaScript.

🔗 **Live App:** https://votenova.onrender.com/

---

## Features

- 🗳️ Create polls instantly
- ➕ Add candidates/options
- ⚡ Real-time voting
- ⏱️ 3-second cooldown timer (anti-spam)
- 📊 View results (vote count based)
- 💾 Save polls
- 🗑️ Delete polls
- 📥 Export results
- 🎯 Clean dashboard UI

---

## Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Hosting:** Render
